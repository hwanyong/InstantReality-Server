# 🤖 서보 설정 가이드 (Servo Config Guide)

각 관절의 물리적 특성을 소프트웨어에 올바르게 반영하기 위한 설정 기준입니다.

---

## 1. Type (관절 종류)
모터가 어떤 방식으로 움직이는지 정의합니다.

| 타입 (Type) | 의미 (Meaning) | 주요 용도 | 예시 |
| :--- | :--- | :--- | :--- |
| **`vertical`** | **상하 회전** (Pitch) | 팔꿈치, 어깨(위아래) | 고개를 끄덕이는 동작 |
| **`horizontal`**| **좌우 회전** (Yaw) | 베이스 회전, 어깨(좌우) | 고개를 좌우로 젓는 동작 |
| **`roll`** | **축 회전** (Roll) | 손목 회전 | 드라이버를 돌리는 동작 |
| **`gripper`** | **그리퍼** (End Effector) | 물체 집기 | 집게 손가락 오므리기 |

---

## 2. Min Pos (0도 위치 기준)
서보의 각도가 `0도` (최소값) 일 때, 로봇이 물리적으로 어떤 상태인지 정의합니다. **회전 방향의 기준점**이 됩니다.

### A. `vertical` 타입일 때
| 값 | 의미 | 0도에서 각도가 커지면(`+`)? |
| :--- | :--- | :--- |
| **`bottom`** | 0도가 **아래쪽** (바닥) | **위로 올라감** (Raise) |
| **`top`** | 0도가 **위쪽** (하늘) | **아래로 내려감** (Lower) |

### B. `horizontal` 타입일 때
| 값 | 의미 | 0도에서 각도가 커지면(`+`)? |
| :--- | :--- | :--- |
| **`left`** | 0도가 **왼쪽** | **오른쪽으로 회전** |
| **`right`** | 0도가 **오른쪽** | **왼쪽으로 회전** |

### C. `roll` 타입일 때 (사용자 시점 기준)
| 값 | 의미 | 0도에서 각도가 커지면(`+`)? |
| :--- | :--- | :--- |
| **`cw`** | 0도가 **최대 시계방향** (Clockwise) | **반시계방향(CCW)으로 풀림** |
| **`ccw`** | 0도가 **최대 반시계방향** (Counter-CW) | **시계방향(CW)으로 감김** |

### D. `gripper` 타입일 때
| 값 | 의미 | 0도에서 각도가 커지면(`+`)? |
| :--- | :--- | :--- |
| **`open`** | 0도가 **열린 상태** | **닫힘** (물체를 잡음) |
| **`close`** | 0도가 **닫힌 상태** | **열림** (물체를 놓음) |

---

## 3. Length (길이)
해당 관절의 회전축 중심으로부터 **"다음 관절의 회전축 중심"** 까지의 직선 거리입니다. (단위: mm)

*   **마지막 관절(손목 등)일 경우**: 관절 중심에서 **끝단(TCP, Tool Center Point)까지의 거리**를 입력합니다.
*   **그리퍼**: 보통 길이가 없거나 손목 길이에 포함되므로 `0`을 입력하는 경우가 많습니다.

---

## 📝 설정 예시 (Example Configuration)

**상황: Base 모터가 0도일 때 왼쪽을 보고 있고, 각도를 키우면 오른쪽으로 180도까지 돌아가는 경우**
```json
{
  "slot_1": {
    "type": "horizontal",  // 좌우 회전하므로
    "min_pos": "left",     // 0도가 왼쪽이므로
    "length": 55.0         // 다음 관절(어깨)까지 높이가 55mm
  }
}
```

**상황: 팔꿈치 관절이 0도일 때 아래로 축 쳐져 있고, 각도를 키우면 팔을 들어 올리는 경우**
```json
{
  "slot_2": {
    "type": "vertical",    // 상하로 움직이므로
    "min_pos": "bottom",   // 0도가 아래쪽이므로
    "length": 105.0        // 다음 관절(손목)까지 길이가 105mm
  }
}
```
